``` bash

```

## setup 

* create needed dir if not present 
```bash
mkdir -p ~/.config/nvim/ && cd ~/.config/nvim/
```

* clone repo
```bash

```

## removal

rm -r ~/.config/nvim 
rm -r ~/.local/share/nvim
